# Paper artifacts

This directory holds the paper-ready figures generated by the experiment
scripts. PDFs are written here automatically by:

* `scripts/run_ablation.py`        -> `figures/ablation_summary.pdf`
* `scripts/run_recovery_curve.py`  -> `figures/recovery_curve.pdf`
* `scripts/run_noise_sweep.py`     -> `figures/noise_sweep.pdf`
* `scripts/run_mef_analysis.py`    -> `figures/merit_order.pdf`,
                                       `figures/mef_heatmap.pdf`

Run `make figures` from the project root to regenerate everything.
